# Playdate/Tetris

This is a school project. We've made tetris (Dedris) in PixiJS.

<a href="https://dedris.netlify.app/">Play the game here!</a>

# Installation

1. Clone this repo.
2. Run npm install.
3. Use a liveserver to start the project.
4. Open index.html in your favourite browser.
